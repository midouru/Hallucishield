# 🛡️ HalluciShield

> AI-powered hallucination detection and verification middleware for Large Language Models.

HalluciShield is a full-stack AI application that detects and verifies factual claims generated by LLMs using Retrieval-Augmented Generation (RAG), vector search, and evidence-backed reasoning.

It sits between an application and an LLM, extracts factual claims, retrieves supporting evidence, verifies each claim, and produces a hallucination score with citations.

---

## ✨ Features

- 🔍 Automatic claim extraction from AI responses
- 📚 Retrieval-Augmented Generation (RAG)
- 🧠 Hybrid evidence retrieval
- 📖 Evidence-backed claim verification
- 📊 Hallucination scoring
- 🔗 Source attribution
- ⚡ Groq-powered LLM reasoning
- ☁️ Chroma Cloud vector database
- 🎨 Modern React dashboard
- 🚀 Fully deployed (Frontend + Backend)

# 🚀 How it Works

1. User submits a prompt.
2. The backend generates an LLM response.
3. HalluciShield extracts factual claims.
4. Claims are normalized.
5. Relevant evidence is retrieved using RAG.
6. Evidence is ranked.
7. Groq verifies each claim.
8. Confidence scores are calculated.
9. Evidence is returned to the frontend.
10. Overall hallucination score is displayed.
