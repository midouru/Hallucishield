# рџ›ҰпёЏ HalluciShield

HalluciShield is an AI middleware layer designed to detect, verify, and reduce hallucinations generated by Large Language Models (LLMs).

Instead of blindly trusting an LLM response, HalluciShield intercepts the generated answer, extracts factual claims, retrieves supporting evidence from a knowledge base, verifies each claim using Retrieval-Augmented Generation (RAG), and produces a confidence-scored audit trail.

---

## рџљЂ Features

### вњ… Query Expansion
Enhances user queries with related terms to improve retrieval quality.

### вњ… Retrieval-Augmented Generation (RAG)
Retrieves relevant facts from a ChromaDB vector database before generating responses.

### вњ… Semantic Search
Uses embeddings to perform similarity-based retrieval rather than keyword matching.

### вњ… Context Reranking
Ranks retrieved documents by relevance before passing them to the LLM.

### вњ… Grounded Answer Generation
Generates responses using verified contextual information.

### вњ… Claim Extraction
Automatically extracts factual claims from generated responses.

### вњ… Claim Verification
Verifies each claim against retrieved evidence from the knowledge base.

### вњ… Confidence Scoring
Assigns confidence scores to every verified claim.

### вњ… Source Attribution
Tracks the source file that supports each verified fact.

### вњ… Audit Trail
Maintains verification metadata including:

- Verification method
- Similarity threshold
- Timestamp
- Supporting evidence

### вњ… Hallucination Score
Calculates the percentage of unsupported claims in the generated response.

## рџ“Ө Installation

### Clone Repository

```bash
git clone <repository-url>
cd HalluciShield
```

### Install Dependencies

```bash

```

### Configure Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
```

### Start ChromaDB

```bash
chroma run
```

### Load Knowledge Base

```bash
npm run ingest
```

### Start Server

```bash

```
