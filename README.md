HalluciShield 🛡️
AI Hallucination Detection Middleware

HalluciShield is a real-time AI middleware designed to detect, verify, and flag hallucinated information generated by Large Language Models (LLMs).

Instead of allowing AI-generated responses to directly reach end users, HalluciShield acts as a proxy layer that:

Intercepts LLM responses
Extracts factual claims
Verifies claims against a trusted knowledge base
Computes confidence scores
Flags potentially hallucinated statements
Provides an audit trail for verification
Problem Statement

Large Language Models often generate responses that sound convincing but contain incorrect or fabricated information.

Examples:

Microsoft acquired GitHub in 2025.
Google was founded in 2010.
Aspirin cures cancer.

These hallucinations can be dangerous in:

Healthcare
Finance
Legal Systems
Enterprise Knowledge Assistants
Customer Support Bots

HalluciShield aims to improve trust and reliability by verifying generated information before it reaches the user.

Features Implemented
Response Generation
Local LLM inference using Ollama
Phi3 / Phi3 Mini integration
Claim Extraction

Extracts structured factual claims from generated responses.

Example:

[
  {
    "claim": "Microsoft acquired GitHub",
    "type": "event"
  }
]
Semantic Verification

Uses embeddings instead of exact text matching.

Example:

Microsoft acquired GitHub

matches:

Microsoft purchases GitHub

through semantic similarity.

Embedding Generation

Current embedding model:

nomic-embed-text

Used to convert text into semantic vectors.

Cosine Similarity Search

Computes similarity between:

Claim Embedding

and

Knowledge Base Embeddings

to find the closest factual evidence.

Confidence Scoring

Example:

{
  "claim": "Microsoft acquired GitHub",
  "verified": true,
  "confidence": 0.94
}
